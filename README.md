#  Weather App

##  Project Description

This is a weather web application built using HTML, CSS, and JavaScript.
It fetches real-time weather data using an API and displays temperature, humidity, wind speed, and weather conditions based on the user's current location.


##  Features

• Real-time weather data using API
• Location-based weather (Geolocation)
• Displays temperature, humidity, and wind speed
• Dynamic weather icons
• Current date display

##  Technologies Used

• HTML
• CSS
• JavaScript
• OpenWeatherMap API


##  Folder Structure

• index.html → Main page
• style.css → Styling
• script.js → Logic & API calls
• assets/icons → Weather icons

##  How to Run

1. Clone or download the repository
2. Add your API key in the script file
3. Open `index.html` in your browser
